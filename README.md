# birthday
A small interactive birthday app built with HTML, CSS, and JavaScript.
checkout: https://nicolep02.github.io/birthday/

## Overview

This project is a short multi-screen experience with the following flow:

start → cat → choose → ending

The main focus is on visuals, interaction, and flow.

## Features

- Multi-screen navigation
- Interactive cat dialogue scene
- Three selectable outcomes
- GIF and PNG asset support
- Custom cursor
- Ending animation effects
- Replay and “choose again” buttons

## Tech Stack

- HTML
- CSS
- JavaScript

No frameworks or external libraries are used.
